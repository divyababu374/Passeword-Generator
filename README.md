🔐 Advanced Password Generator (Task 3)

A secure and user-friendly Password Generator Application built using Python and Tkinter as part of Task 3 of my Python Internship.

This application allows users to generate strong, customizable passwords with multiple advanced features and a clean graphical interface.

🚀 Features

🔢 Custom password length selection

🔤 Character options:

Uppercase letters

Lowercase letters

Numbers

Symbols

🚫 Option to exclude similar characters (O, 0, l, 1)

📊 Password strength indicator (Weak / Medium / Strong)

📋 Copy to clipboard functionality

💾 Save generated passwords to a file

🔄 Reset inputs easily

🌙 Dark / Light mode toggle

🔐 Secure password generation using Python secrets module

🛠️ Technologies Used

Python

Tkinter (GUI Development)

Secrets Module
